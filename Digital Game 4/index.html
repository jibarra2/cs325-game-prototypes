Chicken Farm

  • Originality: What's original about my game is that I've taken the code from the game snake and I've changed the assets so that they are now centered around the theme of chickens.
  • Technical Merit: I wanted to do more with the technicals for the mechanics just didn't really know how to impliment it.
  • Prototype Postmortem: From this prototype the hardest thing was getting my assets to show up on screen. So far everything I've gotten in my code is correct and is in the right places, it just won't show up for some reason. I'm learning a lot more javascript or at least I'm understanding the idea behind certain things I still don't know how to code much of it.
  • Prototype Assets: I made all the assets myself, the chicken is mine and the food can is original as well.
  • Prototype Closest Other Game: Snake is the game that I've modeled my game after and I've used the phaser code for their Snake game.
  • High Concept: Run around as a single chicken collecting food so you can multiply and have your own army of chickens!
  • Theme: I used the "Chicken" card
  • Mandated Variety: For the input we're taking it from the keyboard. We've also got a lot of randomness. For play styles I had collector selected. In terms of genres I would say this is an overhead game.
  • Prototype Goal: I would say this prototype is trying to evaluate the wrap around mechanic where if you go off the edge of the screen, you show up on the other side.
  • Player Experience Goals: I want players to just have fun with the game. I've always loved the simplicity of Snake so I thought reinventing it a bit would be cool.
  • Gameplay: In the game the player has the basic mechanics of moving up, down, left, and right. As you move around the screen you collect food for your chicken to grow the size of the chicken army. But you don't want to have the army collapse in on itself because then you'll have to start ove again!
  • Strategies: The best strategies with the game is to create a long an spacious path when you have a good amount of chickens in your army so that you don't have to worry about hitting yourself to get another piece of food.
  • Story/Setting/Premise: In this world, chickens are starting to become more intellegent so they want to create their own little chicken army to free themselves from the farm they're kept in. They've found a way to clone themselves using the food their Farmer gives them.
  • Target Audience: Anyone who likes Snake, or chickens is really my target audience.
  • Play Time: However long you can survive.
